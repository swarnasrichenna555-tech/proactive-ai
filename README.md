# proactive-ai

A polished AI productivity companion prototype built as a lightweight web app. It helps users:

- capture tasks and deadlines
- prioritize work by urgency and importance
- receive proactive recommendations before deadlines slip
- build simple habits that support momentum
- add tasks by voice when a quick capture is needed

## Run locally

Open the project in a browser via a simple static server:

```bash
cd /workspaces/proactive-ai
python3 -m http.server 8001
```

Then visit http://127.0.0.1:8001/
